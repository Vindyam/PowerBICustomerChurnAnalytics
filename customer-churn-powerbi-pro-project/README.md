# Customer Churn Analytics Dashboard (Power BI)

## Project Overview
This is a **GitHub-ready Power BI portfolio project** built from a customer churn dataset. It is designed to showcase:
- business intelligence dashboard design
- KPI development with DAX
- customer retention analysis
- stakeholder-focused storytelling

## Dataset
- File: `data/customer_churn_data.csv`
- Rows: **1,000**
- Columns: **10**

### Detected fields
- Churn: `Churn`
- Gender: `Gender`
- Contract Type: `ContractType`
- Tenure: `Tenure`
- Monthly Charges: `MonthlyCharges`
- Payment Method: `None`
- Internet Service: `InternetService`

## Suggested Dashboard Pages
### 1. Executive Overview
- Total Customers
- Churned Customers
- Churn Rate
- Average Monthly Charges
- Average Tenure
- Donut chart: Churn vs Retained
- Bar chart: Churn by Contract Type
- Bar chart: Churn by Gender

### 2. Customer Segmentation
- Churn by Payment Method
- Churn by Internet Service
- Churn by Tenure Band
- Segment filters

### 3. Revenue & Risk Analysis
- Monthly Charges by Churn Status
- Monthly Charges vs Tenure scatter plot
- Revenue at risk
- High-risk segment matrix

### 4. Drivers of Churn
- Key Influencers visual
- Decomposition Tree
- Segment comparison table with conditional formatting
